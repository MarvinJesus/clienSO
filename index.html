    Active Directory

    Active Directory (AD) o Directorio Activo (DA) son los términos que utiliza Microsoft para referirse a su implementación de servicio de directorio en una red distribuida de computadores. Utiliza distintos protocolos, principalmente LDAP, DNS, DHCP y Kerberos. De forma sencilla se puede decir que es un servicio establecido en uno o varios servidores en donde se crean objetos tales como usuarios, equipos o grupos, con el objetivo de administrar los inicios de sesión en los equipos conectados a la red, así como también la administrarnistración de políticas en toda la red.

    DNS

    DNS es un protocolo de red, que se utiliza para traducir nombres a otros tipos de datos para el consumo de la computadora, como una dirección IP (un registro A) o cualquiera de varios otros tipos de registros.Prácticamente cada vez que se resuelve un nombre de dominio, ya sea mediante un navegador web o alguna otra fuente, se realiza una consulta DNS. Los servidores DNS de Active Directory suelen ser recursivos y sirven todas las solicitudes DNS para PC dentro del dominio. Los nombres DNS tampoco se resuelven necesariamente en una sola dirección, sino que pueden corresponder a muchos otros registros e incluso pueden referirse a otros registros de forma recursiva utilizando registros como un CNAME.

    DHCP

    El Protocolo de Configuración de Host Dinámica (en inglés: Dynamic Host Configuration Protocol, también conocido por sus siglas: DHCP) es un protocolo de red mediante el cual un servidor asigna dinámicamente una dirección IP y otros parámetros de configuración de red y otros servicioa a cada dispositivo en una red de área local (LAN). Esto hace que los distintos dispositivos puedan comunicarse entre si.

    Servicios

    Active Directory

    Un servicio de directorio, como los Servicios de dominio de Active Directory (AD DS), proporciona los métodos para almacenar datos de directorio y ponerlos a disposición de los usuarios y administradores de la red.

    DHCP

    Sin DHCP, las direcciones IP para computadoras nuevas o computadoras que se mueven de una subred a otra deben configurarse manualmente; Las direcciones IP para las computadoras que se eliminan de la red se deben reclamar manualmente.

    DNS

    Los Servicios de dominio usan DNS como mecanismo de ubicación del controlador de dominio. Cuando se realiza cualquiera de las principales operaciones de Active Directory, como la autenticación, actualización o búsqueda, las computadoras usan DNS para ubicar los controladores de dominio de Active Directory.

    Certificate Authority (CA)

    En criptografía, las expresiones autoridad de certificación, o certificadora, o certificante, o las siglas AC o CA (por la denominación en idioma inglés Certification Authority), señalan a una entidad de confianza, responsable de emitir y revocar los certificados, utilizando en ellos la firma electrónica, para lo cual se emplea la criptografía de clave pública.

    Intranet

    Red informática que utiliza la tecnología del protocolo de Internet para compartir información, sistemas operativos o servicios de computación dentro de una organización. Suele ser interna, en vez de pública como internet, por lo que solo los miembros de esa organización tienen acceso a ella.

    WSUS

    Provee actualizaciones de seguridad para los sistemas operativos Microsoft. Mediante Windows Server Update Services, los administradores pueden manejar centralmente la distribución de parches a través de actualizaciones automáticas a todas las computadoras de la red corporativa.
